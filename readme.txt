== Description ==

Displays recent topics list from IPS Community Suite (former IP.Board) based forum.

**STANDARD FEATURES**

- Displays list of recently updated or created forum topics.
- Fully compatible with IPS Community Suite 4 and higher.
- Follows default forum layout, keeping all colors and functionality.
- SEF URLs to topics, posts, profiles and tags.

**PREMIUM FEATURES**

- Three different layouts: standard, native and compact.
- Display first or last post and all applicable data.
- Display only topics without replies (new topics).
- Exclude or include only certain forums.
- Set number of displayed topics.
- Set number of displayed characters in topic title, forum name, author name or post text.
- Show or hide topic meta, tags or stats.
- Choose the desired date format.
- Show custom text above and below topics list.
- Hide closed topics, define allowed tags in post text and more.

== Installation ==

1. Unpack and upload all files to "/wp-content/plugins/plugin-name" directory. Or simply install plugin through Plugins page.
2. Activate extension on "Plugins" page.
3. Setup plugin according to your liking.
